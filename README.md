📱 Our App – Connecting People & Skilled Workers
📖 Description

Our App is a modern platform built using Flutter that connects people directly with skilled workers in their local area. It removes intermediaries and allows users to easily find, contact, and hire professionals like electricians, plumbers, and carpenters.

The app ensures transparency through ratings, reviews, and clear pricing. With a simple and user-friendly interface, users can quickly access services, while workers can connect with more customers and earn fairly without commissions.

🚀 Features
🔍 Search and find nearby workers
📞 Direct communication
⭐ Ratings and reviews system
💰 Transparent pricing
⚡ Fast and easy booking
🛠️ Tech Stack
Frontend: Flutter
Backend: (Edit panniko – Node.js / Firebase / API)
Database: (MongoDB / Firebase / PostgreSQL)
Containerization: Docker
🐳 Docker Setup

Make sure Docker is installed in your system.

🔧 Build Docker Image
docker build -t our-app .
▶️ Run Container
docker run -p 8080:8080 our-app
📦 Installation (Without Docker)
git clone [https://github.com/manikandan1908/namma-oour]
cd your-repo-name
flutter pub get
flutter run
